**Title extraction logic:** After reading ticket.md content and parsing frontmatter, scan the markdown body for the first heading. Pattern: `# NNNN: Title` (the heading written by the intake agent). Extract the title portion (everything after `NNNN: `). If no heading found, fall back to slug-to-title conversion: `slug.split('-').map(w => w.charAt(0).toUpperCase() + w.slice(1)).join(' ')`.

Create a helper function `extractTitle(body, slug)`:
```javascript
function extractTitle(body, slug) {
  // Try to extract from first markdown heading: # NNNN: Title
  const headingMatch = body.match(/^#\s+\d+:\s+(.+)/m);
  if (headingMatch) {
    return headingMatch[1].trim();
  }
  // Fallback: derive from slug
  return slug.split('-').map(w => w.charAt(0).toUpperCase() + w.slice(1)).join(' ');
}
```
